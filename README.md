# 🎓 KLU Timetable Portal

A student timetable web app built for **KLU,Vijaywada**  
**B.Tech Honours • 2026-27 Semester**

## 🌐 Live Demo
👉 [timetable-portal.vercel.app](https://timetable-portal.vercel.app)

## ✨ Features
- 📅 Weekly timetable with real class schedule
- ✅ Attendance tracker with bunk calculator (75% threshold)
- 🔔 Day-aware class reminders with live countdown
- 📚 Subject & faculty details with OneDrive study material links
- 👤 About page with developer contact links
- 🔄 Multi-page navigation with browser back button support
- 📊 Semester progress bar
